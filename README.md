## 📖 简介

**Topaz V3** 是一个轻量级的智能股票估值分析系统，通过整合 **5 大因子**（价值、质量、动量、波动、红利）对股票进行多维度量化评估，并预测未来 3 个月的收益表现。

系统支持 **A 股**（腾讯财经）和 **美股**（Finnhub API）两大市场的实时数据分析。

---

## ✨ 核心特性

| 特性 | 描述 |
|------|------|
| 🎯 **多因子评分** | 5 大维度量化评分（0-100 分），全面评估股票价值 |
| 🔮 **收益预测** | 基于因子权重预测未来 3 个月收益 |
| ⚠️ **风险评估** | 多维度风险识别，4 级风险等级判定 |
| 💡 **智能建议** | 自动生成买入/持有/回避等投资建议 |
| 🌏 **双市场支持** | A 股 + 美股数据实时获取 |
| 📊 **保守策略** | 针对 A 股/美股采用不同的保守参数 |

## 🚀 快速开始

### 环境要求

- Python 3.7+

### 1. 克隆项目

```bash
git clone <your-repo-url>
cd topaz-v3
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置 API Key

```bash
# 复制环境变量配置模板
cp .env.example .env

# 编辑 .env 文件，填入你的 Finnhub API Key
FINNHUB_API_KEY=your_api_key_here
```

> 🔑 获取 API Key：访问 [https://finnhub.io/](https://finnhub.io/) 注册并获取

### 4. 运行分析

```bash
python3 topaz_analysis_v3_full.py
```

## 📊 因子体系

```
                    ┌─────────────────────────────────┐
                    │      Topaz 多因子评分体系        │
                    └───────────────┬─────────────────┘
                                    │
        ┌───────────┬───────────┬───┴───┬───────────┬───────────┐
        ▼           ▼           ▼       ▼           ▼
    ┌───────┐  ┌───────┐  ┌───────┐ ┌───────┐  ┌───────┐
    │ 价值  │  │ 质量  │  │ 动量  │ │ 波动  │  │ 红利  │
    │ 20%  │  │ 25%  │  │ 20%  │ │ 15%  │  │ 20%  │
    └───────┘  └───────┘  └───────┘ └───────┘  └───────┘
       │           │           │       │           │
    PE/PB      ROE/利润    涨跌幅    日波动     股息率
```

### 因子说明

| 因子 | 权重 | 核心指标 | 评分逻辑 |
|------|------|----------|----------|
| 🏷️ **价值因子** | 20% | PE、PB | 低估值 = 高分 |
| 🏆 **质量因子** | 25% | ROE、利润率、营收增长 | 高盈利 = 高分 |
| 📈 **动量因子** | 20% | 涨跌幅 | 强势上涨 = 高分 |
| 📉 **波动因子** | 15% | 日波动率 | 低波动 = 高分 |
| 💰 **红利因子** | 20% | 股息率 | 高股息 = 高分 |

## 🎯 投资建议逻辑

```
┌─────────────────────────────────────────────────────────────┐
│                    投资决策流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   风险等级判定 ──→ 预测收益计算 ──→ 生成投资建议            │
│                                                             │
│   低风险 + 高预测收益  →  建议买入/强烈推荐                  │
│   中风险 + 中预测收益  →  建议观望/持有                      │
│   高风险/极高风险     →  建议回避                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 建议等级

| 建议类型 | 条件 | 置信度 |
|----------|------|--------|
| 🟢 **建议买入** | 低风险 + 预测收益>25% | 中 |
| 🟡 **可考虑买入** | 低风险 + 预测收益>15% | 中 |
| ⚪ **建议持有** | 低风险 + 预测收益>5% | 中 |
| 🟠 **建议观望** | 中风险 或 低预测收益 | 中 |
| 🔴 **建议回避** | 高风险/极高风险 | 高 |

## 🔧 配置说明

### ⚠️ 安全提示

**切勿将 API Key 硬编码在代码中上传到 GitHub！**

本项目使用 `python-dotenv` 管理敏感配置：

- ✅ `.env` 文件已添加到 `.gitignore`，不会被上传
- ✅ 代码通过 `load_dotenv()` 自动加载 `.env` 中的环境变量
- ✅ 未配置 `.env` 时程序会提示错误

> 💡 **首次运行**：复制 `.env.example` 为 `.env` 并填入你的 API Key

### 获取 Finnhub API Key

1. 访问 [https://finnhub.io/](https://finnhub.io/)
2. 注册免费账号
3. 在 Dashboard 中获取 API Key
4. 复制 `.env.example` 为 `.env` 并填入 API Key
